# 指标采集与指数计算说明

本文档用于说明统计模块中 12 类指标、采集形式、指数计算函数，以及这些数据如何服务于 Top level 管理驾驶舱。

## 一、采集形式代码

| 代码 | 采集形式 | 适合数据 |
|---|---|---|
| A | 系统自动生成 | 打卡时间、项目、次数、达标率、趋势、照片/视频上传率、审核状态 |
| B | 管理员/学校导入 | 学生档案、班级、成绩、课表、作业量、体测数据 |
| C | 教师录入/评价 | 运动态度、课堂表现、请假、体育课占用、伤病记录 |
| D | 学生轻量问卷 | 兴趣、疲劳、压力、睡眠、心情、运动后感受 |
| E | 家长问卷 | 家庭支持、周末运动、补习、通勤、作息和场地条件 |
| F | 设备/传感器 | 心率、步数、GPS、睡眠、配速、运动强度 |
| G | 模型计算 | 指数、相关性、风险预警、趋势预测、综合成长评估 |

## 二、12 类指标采集方式

| 类别 | 推荐采集方式 | 说明 |
|---|---|---|
| 一、学生基础维度 | B 为主，C/E 补充 | 学校、年级、班级、姓名由学校导入；身高体重来自体测或教师录入；住宿、通勤、补习由家长问卷补充。 |
| 二、运动参与度指标 | A 为主，C 补充 | 每日/每周/月运动次数、完成率、达标率、上传率和审核通过率由系统自动生成；请假和特殊缺席由教师确认。 |
| 三、运动质量指标 | A/F 为主，D/C 补充 | 跑步用时、距离、次数来自打卡；心率、配速和强度来自设备；疲劳、热身、拉伸和伤病由问卷或教师记录。 |
| 四、作业与学习压力指标 | B/D/E 采集 | 作业量、完成率、迟交来自作业系统；实际完成时间、难度和压力由学生轻量问卷；补习和娱乐时间由家长或学生补充。 |
| 五、运动积极度指标 | A/D/C/E + G | 主动预约、自愿选择、挑战次数来自系统行为；兴趣、成就感、满意度来自学生问卷；老师和家长提供态度与支持度评价。 |
| 六、学业成绩指标 | B 为主，C/D 补充 | 总分、单科、排名、月考、期中、期末、小测和作业正确率由教务或学习平台导入；课堂表现和学习效率由教师/学生补充。 |
| 七、睡眠与恢复指标 | D/F + G | 入睡、起床、睡眠时长和质量由学生填报或手环采集；运动后睡眠改善、作业压缩睡眠由模型关联计算。 |
| 八、心理与情绪指标 | D 为主，G 预警 | 压力、焦虑、心情、自信、疲惫、社交和体育课感受通过低敏轻问卷收集；低动力趋势只做聚合预警。 |
| 九、家庭与环境指标 | E 为主，D 补充 | 家长支持、家庭运动习惯、作息、周末运动、场地装备、体育培训和体育中考重视度由家长问卷采集。 |
| 十、学校与班级环境指标 | B/C/D + 外部数据 | 班级作业量、课表和体育课占用来自学校系统或教师记录；运动氛围来自学生/教师反馈；天气和空气质量接外部数据。 |
| 十一、关系型指标 | G 模型计算 | 作业量与运动、睡眠与积极度、运动与成绩、考试前运动下降、家长支持与持续运动等关系由多源数据关联分析得出。 |
| 十二、高级指数 | G 综合计算 | 运动参与、积极度、学习负担、睡眠恢复、学业变化、情绪压力、健康风险、班级活跃、年级发展和综合成长指数由模型计算。 |

## 三、指数计算函数

指数函数已落地在：

`src/lib/studentIndexMetrics.ts`

所有指数统一输出 `0-100` 分。正向指数分数越高越好；风险/压力类指数分数越高代表负担或风险越高。

| 函数 | 指数 | 输入数据 | 计算逻辑 |
|---|---|---|---|
| `calculateSportsParticipationIndex()` | 运动参与指数 | 参与率、达标率、月运动次数、连续运动天数、缺席率 | 35% 参与率 + 25% 达标率 + 20% 月频次 + 10% 连续性 + 10% 低缺席 |
| `calculateSportMotivationIndex()` | 运动积极度指数 | 主动选择率、主动预约率、挑战率、兴趣、成就感、最低完成比例 | 行为主动性 + 问卷兴趣/成就感 + 挑战意愿 - 只做最低要求 |
| `calculateLearningLoadIndex()` | 学习负担指数 | 作业时长、补习时长、迟交率、作业压力、睡眠压缩率 | 负担越高分越高，用于识别学习压力风险 |
| `calculateSleepRecoveryIndex()` | 睡眠恢复指数 | 睡眠时长、睡眠质量、白天困倦、入睡时间、运动后恢复 | 接近 8 小时睡眠 + 高质量 + 低困倦 + 不晚睡 + 恢复改善 |
| `calculateAcademicChangeIndex()` | 学业变化指数 | 总分变化、排名提升、作业正确率、小测变化、注意力 | 成绩趋势 + 排名趋势 + 学习过程质量 |
| `calculateEmotionPressureIndex()` | 情绪压力指数 | 学习压力、考试焦虑、疲惫、心情、自信 | 压力越高分越高，心情和自信使用反向计分 |
| `calculateHealthRiskIndex()` | 健康风险指数 | BMI 风险、伤病率、运动限制、高疲劳、心率异常 | 健康风险越高分越高，用于管理层预警 |
| `calculateClassActivityIndex()` | 班级活跃指数 | 班级参与率、集体活动次数、班级氛围、连续性、项目覆盖 | 班级层面的运动活跃与组织健康度 |
| `calculateGradeSportsDevelopmentIndex()` | 年级体育发展指数 | 年级参与率、达标率、项目覆盖、提升率、班级均衡度 | 年级体育发展水平和均衡程度 |
| `calculateStudentGrowthIndex()` | 学生综合成长指数 | 所有核心指数 | 正向指数加权 + 学习负担/压力/健康风险反向计分 |
| `calculatePearsonCorrelation()` | 关系型指标相关性 | 两组时间序列，例如每日作业量与每日运动次数 | 输出 -1 到 1 的相关系数，用于分析变量之间的关系 |

## 四、各指数输入字段建议

### 1. 运动参与指数

输入字段：

- `participationRate`：参与率，来自打卡系统
- `passRate`：达标率，来自打卡结果
- `monthlyExerciseTimes`：月运动次数，来自打卡次数
- `continuityDays`：连续运动天数，系统计算
- `absenceRate`：缺席率，系统记录或教师确认

采集方式：A + C + G

### 2. 运动积极度指数

输入字段：

- `activeChoiceRate`：主动选择运动比例
- `activeBookingRate`：主动预约比例
- `challengeRate`：挑战高目标比例
- `interestScore`：运动兴趣自评
- `achievementScore`：运动后成就感
- `minimumOnlyRate`：只完成最低要求比例

采集方式：A + D + C + E + G

### 3. 学习负担指数

输入字段：

- `homeworkMinutes`：每日作业分钟数
- `tutoringHoursPerWeek`：每周补习小时数
- `lateHomeworkRate`：作业迟交率
- `homeworkPressureScore`：作业压力自评
- `sleepCompressionRate`：作业压缩睡眠比例

采集方式：B + D + E + G

### 4. 睡眠恢复指数

输入字段：

- `sleepHours`：睡眠时长
- `sleepQualityScore`：睡眠质量评分
- `daytimeSleepinessScore`：白天困倦程度
- `bedtimeHour`：入睡时间
- `recoveryImprovementRate`：运动后恢复改善率

采集方式：D + F + G

### 5. 学业变化指数

输入字段：

- `totalScoreDelta`：总分变化
- `rankImprovement`：排名提升
- `homeworkAccuracyRate`：作业正确率
- `quizScoreDelta`：小测成绩变化
- `attentionScore`：注意力评价

采集方式：B + C + D + G

### 6. 情绪压力指数

输入字段：

- `stressScore`：学习压力
- `examAnxietyScore`：考试焦虑
- `fatigueScore`：疲惫感
- `moodScore`：心情评分
- `confidenceScore`：自信心

采集方式：D + G

### 7. 健康风险指数

输入字段：

- `bmiRiskRate`：BMI 风险比例
- `injuryRate`：伤病发生比例
- `restrictedRate`：运动限制比例
- `highFatigueRate`：高疲劳比例
- `abnormalHeartRateRate`：心率异常比例

采集方式：B + C + F + G

### 8. 班级活跃指数

输入字段：

- `participationRate`：班级参与率
- `groupActivityTimes`：集体活动次数
- `classAtmosphereScore`：班级运动氛围评分
- `continuityDays`：连续运动天数
- `projectCoverageRate`：项目覆盖率

采集方式：A + C + D + G

### 9. 年级体育发展指数

输入字段：

- `participationRate`：年级参与率
- `passRate`：年级达标率
- `projectCoverageRate`：项目覆盖率
- `improvementRate`：整体提升率
- `classBalanceRate`：班级均衡度

采集方式：A + G

### 10. 学生综合成长指数

输入字段：

- `sportsParticipationIndex`
- `sportMotivationIndex`
- `learningLoadIndex`
- `sleepRecoveryIndex`
- `academicChangeIndex`
- `emotionPressureIndex`
- `healthRiskIndex`
- `classActivityIndex`
- `gradeDevelopmentIndex`

采集方式：G

说明：

- 运动参与、运动积极度、睡眠恢复、学业变化、班级活跃、年级发展是正向输入。
- 学习负担、情绪压力、健康风险是反向输入。
- 综合成长指数不是简单平均，而是加权综合。

## 五、关系型指标函数

关系型指标使用：

`calculatePearsonCorrelation(xValues, yValues)`

适合分析：

- 作业量与运动参与度关系
- 睡眠不足与运动积极度关系
- 运动参与度与成绩变化关系
- 考试前两周运动量下降
- 家长支持与持续运动关系
- 不同运动项目对学习状态的影响

输出解释：

| 相关系数 | 含义 |
|---|---|
| 接近 1 | 强正相关 |
| 接近 0 | 关系不明显 |
| 接近 -1 | 强负相关 |

示例：

```ts
const correlation = calculatePearsonCorrelation(
  [120, 150, 180, 210], // 每日作业分钟数
  [4, 3, 2, 1], // 每日运动次数
);
```

如果结果接近 `-1`，说明作业量增加时，运动参与明显下降。

## 六、产品落地建议

1. 第一阶段只采集低风险数据：学生基础、项目、打卡、达标、成绩导入。
2. 第二阶段加入轻量问卷：兴趣、疲劳、压力、睡眠、运动后感受。
3. 第三阶段加入家长问卷和学校教务数据：补习、作业、成绩、课表。
4. 第四阶段再考虑设备数据：心率、步数、睡眠、GPS。
5. 管理层优先看聚合数据，不默认展示个体敏感心理/健康数据。
6. 风险类指数只做预警，不直接给学生贴标签。
